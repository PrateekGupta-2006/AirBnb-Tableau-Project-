#  Airbnb Tableau Dashboard Analysis

##  Project Overview
This project analyzes Airbnb listing data to uncover insights related to pricing, availability, location trends and customer preferences. The analysis is visualized using an interactive Tableau dashboard.


##  Objectives
- Understand pricing trends across different locations
- Identify high demand areas
- Analyze availability patterns
- Provide insights for hosts and business decision making

##  Tools & Technologies
- Tableau
- Microsoft Excel
- CSV Dataset



## Dataset
- Source: Airbnb dataset (public dataset)
- Includes:
  - Listing price
  - Location
  - Availability
  - Reviews
  - Room type

## Data Cleaning
- Removed null values
- Handled missing price entries
- Standardized column names
- Filtered irrelevant columns

## Key KPIs
- Average Price per Location
- Total Listings
- Availability (Days)
- Number of Reviews
- Price Distribution


##  Dashboard Features
- Interactive filters (location, room type)
- Price comparison charts
- Availability heatmaps
- Review analysis


##Key Insights
- Certain locations have significantly higher pricing
- Entire homes/apartments generate higher revenue
- Listings with more reviews tend to have better occupancy
- Availability impacts pricing strategy


##  How to Use
1. Download the `.twbx` file from the dashboard folder
2. Open in Tableau Desktop / Tableau Public
3. Explore filters and insights
   

##  Future Improvements
